# Introduction

SSR Fetch based on Next JS.

## Installation

Use npm or Yarn

```bash
$ npm install ssr-fetch
```

```bash
$ yarn add ssr-fetch
```

## Example

```ts
// here the base url <url>/api

let login: any = await rPost('/login', { email, password })

// so after the use the ssr-fetch package the post request use like
// <url>/api/login
```
